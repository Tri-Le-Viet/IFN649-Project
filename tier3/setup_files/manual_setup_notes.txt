Remember: need to keep inbound ports 80 and 8883 open

create mosquitto.conf and my_server files

add line "include /etc/nginx/sites-enabled/*;" to http block of /etc/nginx/nginx.conf
copy my_server to /etc/nginx/sites-available

run the following to symlink my-server to sites-enabled:
  sudo ln -s /etc/nginx/sites-available/my-server /etc/nginx/sites-enabled/

then:
  sudo nginx -s reload

then start gunicorn:
gunicorn -w 4 -b 127.0.0.1:5000 app:gunicorn_app
